Программа заходит на сайт lun.ua и с определенным инервалом ищет квартиры с указанным фильтром и при нахождении отправляет ссылки на них на почту. Программа будет работать
корректно только если выбран фильтр "Только актуальные за сегодня"

1) Распаковать архив куда угодно.
2) В архиве есть файл constants.json. В нем указываются настройки для программы. Нужно проставить настройки
Объяснение настроек:

{
  "urls": [
    {
      "link": "https://www.lun.ua/аренда-квартир-киев?areaTotalMin=45&currency=2&priceMax=12000",  // ссылка, которую нужно скопировать после выбора всех необходимых фильтров на сайте lun.ua и нажатия кнопки "Найти"
      "name": "любое название"   // название для удобства, будет приходить в заголовке письма для этого фильтра.
    },
    {
      "link": "https://www.lun.ua/аренда-квартир-киев?areaTotalMin=45&currency=2&priceMax=16000", // ссылка с фильтром 2
      "name": "любое название"  // название для фильтра 2
    }, // можно добавлять через запятую сколько угодно таких конструкций.
    ...
  ],
  "interval": 600000,   // C какой переодичностю делать запросы. По умолчанию 60000 миллисекунд, что равно 10 минутам.
  "recipients": [
    "testemail@test.test", //адрес получателя писем с квартирами 1
    "testemail2@test.test" // адрес получателя писем с квартирами 2
    ...
  ],
  "sendingEmail": "lunupdater@gmail.com", //отправитель писем. Должен быть зарегестрированный почтовый ящик на gmail.com
  "sendingPassword": "c3td^e6QFPC+nczN+ZA&Jff^+bY%N6" // пароль от ящика.
}


3) Зайти на этот сайт и скачать Node https://nodejs.org/en/download/
4) Установить Node и перезагрузить компьютер.
5) Два раза кликнуть на файл start.bat в распакованном архиве
6) Не выключать программу и компьютер.
